codeforces #398 c

题意说有一棵树，每个点有一个权值，求两个割边把这个树分成权值相等的三部分。

就是tree-dp维护每个点的子树权值和，那么答案可以分成两种情况。

  1. 两个割边在两个点上方，其中一个不是另一个的祖先或者子孙。两棵树的权值都是sum/3
  2. 两个割边在祖先和子孙的上方，权值分别是sum*2/3和sum/3。

分情况搞就好了，注意第二种情况在sum = 0的情况下要判断祖先是不是根节点。
